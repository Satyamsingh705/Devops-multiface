**CHAPTER 1: INTRODUCTION**

(First page of the Introduction chapter — page numbering will start at 1 here when converting to Word.)

This project implements a role-based face recognition attendance system built with FastAPI and InsightFace. The application supports multiple roles (admin, faculty, trainer, security) and includes a full observability stack (Prometheus, Grafana, Alertmanager) for local development and Kubernetes deployment.

Key features:
- Face encoding and recognition via InsightFace
- SQLite-backed embeddings storage with lightweight schema migration
- Role-based UI and session handling
- Prometheus metrics endpoint and Grafana dashboards
- Docker Compose and Kubernetes manifests for full-stack deployment
